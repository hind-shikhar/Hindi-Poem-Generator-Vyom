# Hindi Poem Generator

An AI-powered web application that generates **creative Hindi poems using Natural Language Processing (NLP) and deep learning models**.
The system takes a user prompt or theme and produces poetic text in Hindi, demonstrating the ability of modern language models to generate meaningful literary content.

---

## Live Demo

Hugging Face Space:
https://huggingface.co/spaces/welcometoml/hindi_poem_generator_Vyom

---

## Project Overview

This project explores **AI-based text generation** for creative writing in Hindi.
The model is trained or configured to generate poetic lines based on user prompts, making it possible to automatically create poetry in different themes and styles.

The application is deployed as an interactive web interface where users can easily generate poems with a single click.

## Installation

Clone the repository

git clone https://github.com/hind-shikhar/hindi-poem-generator.git

Move to the project folder

cd hindi-poem-generator

Install dependencies

pip install -r requirements.txt

Run the application

python app.py

---

## How It Works

1. The user enters a **theme or prompt**.
2. The NLP model processes the input.
3. The language model generates poetic text in Hindi.
4. The generated poem is displayed through a web interface.
